# Programa de registros de tipos de café ☕🍪
# Lenguaje de programación Python ¬_¬
## Descripción: 😉
El proyecto consiste en gestionar un sistema para registrar los tipos de café en una
cafetería llamada AromaCampus cada clase tiene una funcionalidad diferente, ademas se creó un archivo tipo markdown donde se registrarán los tipos de café.
## ¿Qué soluciona? 🤨
Esta aplicación ayuda a las cafeterias que tiene problemas con el registro de sus tipos de café, es una pequeña ayuda para que se agilice todo en la cafeteria.
## Funcionamiento y estructura: 😃
En cada una de las clases existe una funcionalidad distinta, cada una se encarga de una parte del codigo,
### Agregar.py: 
```
'''
En esta rama se maneja la opción para agregar  un café
usamos un input para preguntarle a la persona que tipo de café desea
'''
print=('*** Bienvenido ***')
cafe=str(input('Porfavor ingrese el tipo de café que desea: '))

```
En esta parte del codigo se le pide al usuario que registre el café que desea agregar.

### Editar.py:
```
print("Bienvenido al sistema para editar cafes ya existentes")
cafe=str(input('Ingrese porfavor el nombre del café a editar: '))
nombre=str(input('Usted desea cambiar el nombre, porfavor digite el nuevo nombre: '))
print("El nuevo nombre del cafe es: ",nombre )
```
Esta parte del codigo es la encargada de modificar el nombre del tipo de café, primeor se le pregunta al usuario el nombre del café que desea cambiar, luego se le pide que digite el nuevo nombre y al final le muestra al usuario como quedó el nombre que cambió.

### Eliminar.py:
```
print("Bienvenido a la clase para eliminar café")
nombre=str(input("Ingrese el nombre del café a eliminar: "))
print("Se ha eliminado el cafe con el nombre: ", nombre)
```
Esta parte del codigo es la encargada de elimar el tipo de café, primero le pide al usario que ingrewse el nombre del café a eliminar y luego de su eliminación le muestra al usario el nombre del café que se eliminó.

### Productos.md:

Aquí se guardan los tipos de café, este documento usa estructura MarkDown y su texto genera una tabla que se veria así:

|----|----|----|----|
| Café | Solo cafe | 5000 | Disponible |
| Café con leche | Es cafe con leche | 6000 | Disponible |

| Espresso | Basicamente un café | 8000 | Disponible |
